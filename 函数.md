## 函数

> 可接受输入，执行指令并返回输出的复合语句。

**调用（ call）**一个函数，意味着为函数提供执行指令并返回输出所需的输入。函数的
每一个输入就是一个**参数（ parameter）**。当你为函数提供参数时，则被称为“函数传参”。 

**定义函数**

> 创建一个函数，需要选择函数名，并定义其参数、行为和返回值。 

```python
def [函数名]([参数]):
	[函数定义]
    return [结果]					# 没有return则会返回None
```

#### 文档字符串

> 解释函数的功能，记录所需的参数类型。

```python
def add(x, y):
    """
    返回 x + y 的值
    :param x: int.
    :param y: int.
    :return: int, x 与 y 之和
    """
    return x + y
```
