## 容器

列表、元组和字典

字符串列表和元组都是**可迭代的**（iterable）。

如果可以使用循环访问对象中的每一个元素，那么该对象是可迭代的，被称为**可迭代对象**。

#### 列表

[]表示

索引从0开始

```python
[列表名][[索引]]
```

获取不存在的索引报异常

列表是可变的

不能对空列表使用pop方法

#### 元组

()表示

元组不可变

逗号隔开，即使只有一个元素，也要有逗号

```python
# 创建元组
my_tuple = tuple()
mytuple = ()
```

#### 字典

映射、键值对

字典的键是无序的，不可变的

#### 字符串操作

三引号可以跨行

可迭代，支持索引

不可变
